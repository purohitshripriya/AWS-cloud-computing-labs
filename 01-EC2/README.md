# Amazon EC2 Lab

## Overview

Amazon Elastic Compute Cloud (EC2) is a web service that provides secure and resizable virtual servers in the cloud. It allows users to launch and manage virtual machines for hosting applications and services.

---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

## Objective

- Launch an Amazon EC2 instance.
- Configure a Security Group.
- Create and use a Key Pair.
- Connect to the instance using SSH (PuTTY).
- Understand the basic workflow of EC2 deployment.

---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

## AWS Services Used

- Amazon EC2
- Security Groups
- Key Pair
- Elastic IP (Optional)

--------------------------------------------------------------------------------------------------------------------------------------------------------------------

## Steps Performed

1. Logged in to the AWS Management Console.
2. Created a new EC2 instance.
3. Selected **Amazon Linux 2023** as the AMI.
4. Chose the **t2.micro** instance type.
5. Configured the Security Group to allow:
   - SSH (Port 22)
   - HTTP (Port 80)
6. Connected to the EC2 instance using PuTTY.
7. Verified successful login through the terminal.

--------------------------------------------------------------------------------------------------------------------------------------------------------------------

## Outcome

Successfully launched and connected to an Amazon EC2 instance on AWS.
